Project Overview
I built a Markov model to compare Standard Therapy vs. a hypothetical New Therapy for Type 2 Diabetes. The model calculates costs, QALYs, and ICER over a 20-year time horizon.

Key components included:

Health States: Controlled, Complication, Death

Transition Matrix: Simulates patient movement between health states

Markov Trace: Tracks cohort distribution over cycles

Cost & QALY Calculation: Computes total costs and quality-adjusted life years

Incremental Cost-Effectiveness Ratio (ICER): Evaluates cost per additional QALY
